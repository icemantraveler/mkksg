Note: All commands are assuming your character is facing right.
If you are facing left, you must reverse the directional commands.
Backwards becoming Forwards and Vice Versa.
It's recommended that your controls match the default setting.
Key:
U = Up/Jump
D = Down/Crouch/Low Block
F = Forward/Toward
B = Back/Away/Block		 Dash: F, F
Back Dash: B, B
Uppercut: (D+Δ)
Sweep: (D+X)
Pause/Movelist: START
Universal:
L = Light Attack
M = Medium Attack
H = Heavy Attack
CP = "*Character Power"
TH = **Throw
SI = Stage Interaction
FS = Flip Stance
MB = Meter Burn
SM = "Super Move"		 PlayStation 4:
L = □
M = Δ
H = X
CP = O
TH = L1/(□+X)
SI = R1/(□+Δ)
FS = L2
MB = R2
SM = (L2+R2)/(Δ+X+R2)		    
*Weapon Switch/Gadget Deploy/Throw depending on character
** Forward Throw = (F+L1)/(F+□+X)
** Reverse Throw = (B+L1)/(B+□+X)
** Throw Reversal = L1
Level Transition = (B+X)
Clash Breaker = (F+R2) (During Combo, when you're on your second red health bar)
Block Escape = R2 immediately after blocked move (Costs 1 Meter)
Bounce Cancel = F, F/B, B, R2 (Costs 2 Meters) (During Combo)
Attack Counter = Special Move after Blocked Attack
Wakeup Attack = Special Move as you rise from being knocked down
Tech Roll = □, Δ, X or O when hitting ground
