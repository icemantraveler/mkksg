It's recommended that your controls match the default setting.
Key:
U = Up/Jump
D = Down/Crouch
F = Forward/Toward
B = Back/Away
Universal:
P = Punch
K = Kick
BL = Block
RN = Run		 Game Boy Advance:
P = B
K = A
BL = R
RN = L
Close Quarters:
Throw: P (Close)
Knee: K (Close)
Crouched Moves:
Crouched Block: (D+BL)
Crouched Punch: (DF+P)
Uppercut: (D+P)
Crouched Low Kick: (D+K)
Crouched High Kick: (DF+K)
Spin Moves:
Foot Sweep: (DB+K)
Roundhouse Kick: (B+K)
Aerial Moves:
Flying Punch: (U/UF/UB+P)
Flying Kick: (U/UF/UB+K)
Finisher Distances Explained:
Anywhere: Distance doesn't matter, can be any range (outside sweep or further works best).

Note: I wouldn't bother with holding BL during a Finisher, just input the button combination, regardless if the character is jumping around, the Finisher should still activate correctly.
