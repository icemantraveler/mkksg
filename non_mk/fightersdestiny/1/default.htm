Note: All commands are assuming your character is facing right.
If you are facing left, you must reverse the directional commands.
Backwards becoming Forwards and Vice Versa.
It's recommended that your controls match the default setting.
Key:
= Up
= Down
= Back/Block
= Forward
= Down/Back/Low Block
= Down/Forward
= Up/Back
= Up/Forward
= Dark arrows indicate the direction must be tapped
= White arrows indicate the direction must be charged or held back for a certain amount of time
= Repeatedly tap a button for brief period of time
= Hold a button for a brief period of time
= Release a button after holding for a certain amount of time
Arcade:
LP = Light Punch
MP = Medium Punch
HP = Heavy Punch
LK = Light Kick
MK = Medium Kick
HK = Heavy Kick		  

P = Any Punch
K = Any Kick
2P = Any 2 Punch Buttons Simultaneously
2K = Any 2 Kick Buttons Simultaneously
3P = (LP+MP+HP)
3K = (LK+MK+HK)
