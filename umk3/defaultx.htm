Note: All commands are assuming your character is facing right.
If you are facing left, you must reverse the directional commands.
Backwards becoming Forwards and Vice Versa.
It's recommended that your controls match the default setting.
Key:
U = Up/Jump
D = Down/Crouch
F = Forward/Toward
B = Back/Away
Universal:
HP = High Punch
LP = Low Punch
HK = High Kick
LK = Low Kick
BL = Block
RN = Run		 Super Nintendo:
HP = Y
LP = B
HK = X
LK = A
BL = L Trigger
RN = R Trigger		 Sega Genesis 3 Pad:
HP = (A+B)
LP = A
HK = (B+C)
LK = C
BL = B
RN = START		 Sega Genesis 6 Pad:
HP = X
LP = A
HK = Z
LK = C
BL = B
RN = Y
Babalities & Friendships: You must not press BL in the final round.
Mercy: (Hold RN), D, D, D (Release RN) Must be executed at the end of the third round.  (SNES ONLY)
Stage Fatalities available: The Subway, Bell Tower, Pit 3, Scorpion's Lair.

Fighting (Human) Smoke:
Input the code on the VS screen. Player 1 and Player 2 must put in the correct code respectively:
P1: 2-0-5
P2: 2-0-5
Playing as (Human) Smoke:
SNES: Choose Cyber-Ninja Smoke as a character. Hold (B+HP+HK+BL+RN) before the "Fight" screen appears or between rounds. Cyber-Ninja Smoke will explode and change into Human Smoke.

GENESIS: Choose Cyber-Ninja Smoke as a character. Hold (LP+HP+LK+HK) before the "Fight" screen appears or between rounds. Cyber-Ninja Smoke will explode and change into Human Smoke.
