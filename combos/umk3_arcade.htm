Key:
U = Up/Jump
D = Down/Crouch
F = Forward/Toward
B = Back/Away
Universal:
HP = High Punch
LP = Low Punch
HK = High Kick
LK = Low Kick
BL = Block
RN = Run		 Xbox 360:
HP = X
LP = A
HK = Y
LK = B
BL = RT
RN = LT		 PlayStation 2/3:
HP = □
LP = X
HK = Δ
LK = O
BL = L1/R1
RN = L2/R2		 Sega Saturn:
HP = X
LP = A
HK = Z
LK = C
BL = B/L Trigger
RN = Y/R Trigger
Note: Most combos that start with HK can also be started with LK and vice versa.
