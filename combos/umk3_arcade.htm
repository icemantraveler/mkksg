Note: All commands are assuming your character is facing right.
If you are facing left, you must reverse the directional commands.
Backwards becoming Forwards and Vice Versa.
It's recommended that your controls match the default setting.
Key:
U = Up/Jump
D = Down/Crouch
F = Forward/Toward
B = Back/Away
Universal:
HP = High Punch
LP = Low Punch
HK = High Kick
LK = Low Kick
BL = Block
RN = Run		 XBOX 360:
HP = X
LP = A
HK = Y
LK = B
BL = R Trigger
RN = L Trigger		 PlayStation 2/3:
HP = □
LP = X
HK = Δ
LK = O
BL = L1/R1
RN = L2/R2		 Sega Saturn:
HP = X
LP = A
HK = Z
LK = C
BL = B/L Trigger
RN = Y/R Trigger
Note: Most combos that start with HK can also be started with LK and vice versa.
