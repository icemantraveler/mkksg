Combos - Xbox 360/PlayStation 3 (Universal)

You can e-mail corrections/additions to The Webmaster
Last updated: 02/16/25
Note: All commands are assuming your character is facing right.
If you are facing left, you must reverse the directional commands.
Backwards becoming Forwards and Vice Versa.
It's recommended that your controls match the default setting.
Key:
U = Up/Jump
D = Down/Crouch
F = Forward/Toward
B = Back/Away
Pause/Movelist: Start
Universal:
1 = Attack 1
2 = Attack 2
3 = Attack 3
4 = Attack 4
TH = Throw/(1+3)
*CK = Close Kombat/(2+4)
3D= 3D Movement
BL = Block
Evade = (1+2)
Rage Mode = (3D+BL)
2D Movement = D-Pad		 Xbox 360:
1 = X
2 = Y
3 = A
4 = B
TH = LB/(X+A)
*CK = RB/(Y+B)
3D = (LT+U/D) on D-Pad
BL = RT
Evade = (X+Y)
Rage Mode = (LT+RT)
2D Movement = D-Pad		 PlayStation 3:
1 = □
2 = Δ
3 = X
4 = O
TH = L1/(□+X)
*CK = R1/(Δ+O)
3D = (L2+U/D) on D-Pad
BL = R2
Evade = (□+Δ)
Rage Mode = (L2+R2)
2D Movement = D-Pad
*Crouching/Ducking while in Close Kombat = Low Close Kombat
