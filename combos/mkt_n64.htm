It's recommended that your controls match the default setting.
Key:
U = Up/Jump
D = Down/Crouch
F = Forward/Toward
B = Back/Away
Universal:
HP = High Punch
LP = Low Punch
HK = High Kick
LK = Low Kick
BL = Block
RN = Run		 Nintendo 64:
HP = B
LP = A
HK = C▲
LK = C►
BL = C◄/R
RN = C▼/L
* = Difference between PS and N64 Versions of MKT
Note: Most combos that start with HK can also be started with LK and vice versa.
