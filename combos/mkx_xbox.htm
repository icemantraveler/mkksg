Universal:
FP = Front Punch
BP = Back Punch
FK = Front Kick
BK = Back Kick
*TH = Throw (Away)
*(F+TH) = Throw (Towards)
SI = Stage Interact
FS = Flip Stance
BL = Block

TH (Towards) Reversal = BP/BK		 XBOX One:
FP = X
BP = Y
FK = A
BK = B

SI = RB
FS = LT
BL = RT
