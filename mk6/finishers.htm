Note: All commands are assuming your character is facing right.
If you are facing left, you must reverse the directional commands.
Backwards becoming Forwards and Vice Versa.
It's recommended that your controls match the default setting.
Key:
U = Up
D = Down
F = Forward/Toward
B = Back/Away
Universal:
1 = Attack 1
2 = Attack 2
3 = Attack 3
4 = Attack 4
BL = Block
CS = Change Style
TH = Throw
GW= Grab Weapon		 Xbox:
1 = X
2 = Y
3 = A
4 = B
BL = RT
CS = LT
TH = (BLACK)
GW = (White)		 PlayStation 2:
1 = □
2 = Δ
3 = X
4 = O
BL = R2
CS = L1
TH = R1
GW = L2		 GameCube:
1 = B
2 = Y
3 = A
4 = X
BL = R
CS = L
TH = Z
GW = C-Stick
