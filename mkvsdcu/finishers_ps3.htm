Finishers - PlayStation 3

Last updated: 02/16/25
Note: All commands are assuming your character is facing right.
If you are facing left, you must reverse the directional commands.
Backwards becoming Forwards and Vice Versa.
It's recommended that your controls match the default setting.
Key:
U = Up/Jump
D = Down/Crouch
F = Forward/Toward
B = Back/Away
Pause/Move List: START
Universal:
1 = Attack 1
2 = Attack 2
3 = Attack 3
4 = Attack 4
TH = Throw/(1+3)
*CK = Close Kombat/(2+4)
3D = 3D Movement
BL = Block
Evade = (1+2)
Rage Mode = (3D+BL)
2D Movement = D-Pad		 PlayStation 3:
1 = □
2 = Δ
3 = X
4 = O
TH = L1/(□+X)
*CK = R1/(Δ+O)
3D = (L2+U/D) on D-Pad
BL = R2
Evade = (□+Δ)
Rage Mode = (L2+R2)
2D Movement = D-Pad
*Crouching/Ducking while in Close Kombat = Low Close Kombat

Fatality/Heroic Brutality Distances Explained:
Close: Right next to your opponent.
Sweep: A couple of steps away (close enough to land a sweep).

Note: When (Hold R2) is shown in italics for Fatality inputs, it means that holding R2 is optional.
The Fatality will still work without holding R2, though it may be easier if you do.
Just remember to release R2 before pressing the final button in the Finisher combo.
